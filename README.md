## Welcome to Brain Games!

Brain Games is vanilla JS-project, consists of five console brain games: even, calculator, gcd, progression, prime numbers

### Setup:

```
git clone https://github.com/mishken3/frontend-project-lvl1
npm install
```

---

### Prime games launch:

```
make brain-even
make brain-calc
make brain-gcd
make brain-progression
make brain-prime
```

### Alternative games launch

Also you can install games as a local-packages and launch them from root foled of project: `npm link ` and after `brain-*game`

```
npm link
brain-even
```

## Games

The goal is to answer "yes" if given number is even, otherwise answer "no".
